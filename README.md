### 👨‍💻 About Me

I'm an **AI & Data Engineer** who genuinely loves to play with AI and Data.  
Whether it's wrangling messy datasets, building intelligent pipelines, or experimenting with the latest AI models — I'm always having fun with it.

- 🤖 **AI Engineering** — LLMs, RAG systems, embeddings, and intelligent automation
- 📊 **Data Engineering** — ETL/ELT pipelines, data modeling, and data quality
- ⚡ **End-to-End Workflows** — from raw data ingestion all the way to AI-powered insights
- 🔍 **Semantic Search & Retrieval** — vector databases, FAISS, and smart retrieval systems
- 🛠️ **Backend APIs** — FastAPI & Flask for serving AI and data products

I believe the best way to learn is to **build things and break things**.  
If there's data involved or AI can help — I'm already interested.

💡 *"Play with data, ship with purpose."*

### 🎯 What I'm Working On

- 🔗 **RAG Pipelines** — building reliable, production-ready retrieval-augmented generation systems
- 🗄️ **Data Pipelines** — designing scalable ETL workflows and efficient data architectures
- 🤖 **LLM Experiments** — fine-tuning, prompt engineering, and local model deployments
- 📈 **Analytics & Visualization** — turning data into stories with Tableau and Power BI
- ☁️ **MLOps** — model monitoring, versioning, and automated retraining workflows
